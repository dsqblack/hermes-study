# Music Player v2.4

独立运行的 PyQt5 桌面音乐播放器，Spotify 深色风格。双击即用，无需任何依赖。

## 快速启动

```
D:\ai_workspace\hermes\music\music-exe\
└── Music-player.exe              ← 双击运行
```

首次启动自动扫描 `D:\ai_workspace\hermes\music\` 目录下的音频文件。

## 功能列表

### 播放控制

| 操作 | 位置 | 说明 |
||---|---|---|
| ▶ 播放 | 左栏绿色圆按钮 | 开始播放当前选中曲目 |
| ⏸ 暂停 | 左栏绿色圆按钮 | 同一按钮，播放时变为暂停 |
| ⏮ 上一首 | 左栏灰色圆按钮 | 切回上一首 |
| ⏭ 下一首 | 左栏灰色圆按钮 | 跳到下一首 |
| 🔀 随机播放 | 左栏灰色圆按钮(⏮旁边) | 随机打乱当前列表顺序并继续播放，按钮闪烁绿色反馈 |
| 点击曲目 ▶ | 列表行右侧 | 直接播放该曲目 |
| 拖动进度条 | 左栏播放器内 | 跳转到任意位置 |
| 拖动音量 VOL | 左栏底部 | 调节音量 0-100 |

### 推荐与列表

| 操作 | 位置 | 说明 |
|---|---|---|
| 今日推荐 | 右栏顶 | 查看今日推荐缓存，不刷新不重算 |
| ↻ 刷新 | 右栏顶 | 重新扫描曲库 + 同时刷新今日推荐和探索 |
| 🔍 探索 | 右栏顶 | 展示缓存探索列表（不重算），点↻刷新后再看 |
| ALL | 右栏顶 | 重新扫描曲库，查看所有曲目 |
| ♥ 喜欢 | 曲目行右侧 | 标记喜欢，按钮变红闪烁恢复 |
| ⏭ 跳过 | 曲目行右侧 | 跳过该曲目 |
| 📁 选目录 | 顶栏右侧 | 弹窗选择音乐文件夹，切换曲库 |

### 推荐系统

```
每首歌得分 = 标签匹配分 + 新鲜度加分 - 近期疲劳扣分

标签匹配分:  你喜欢的艺人权重之和（like +1, skip -0.5）
新鲜度加分:  没听过的歌 +0.3
疲劳扣分:    一周内每播一次 -0.3
多样性限制:  同艺人最多占2首
```

推荐理由显示在歌名旁边（亮绿色文字），每次点↻刷新会随机打乱同分组内顺序。

## 支持格式

- `.ogg` — OGG Vorbis（QQ音乐下载常见格式）
- `.mp3` — MPEG Audio
- `.flac` — Free Lossless Audio Codec
- `.mflac` — QQ音乐加密FLAC（自动解密播放）
- `.wav` — Waveform Audio
- `.m4a` — MPEG-4 Audio
- `.wma` — Windows Media Audio

### MFLAC 加密文件

`.mflac` 文件是QQ音乐下载的加密FLAC，播放器启动时自动检测并用 QMCv2 算法解密为临时 FLAC 播放。解密后的文件不会保存到磁盘，关闭播放器自动清理。

## 粒子动画效果

封面区域在播放音乐时显示彩色粒子动画：

- 粒子从中心向四周爆发，颜色随机从绿/青/紫/粉红循环
- 能量模拟节拍律动，能量高时粒子更多、更大、速度更快
- 暂停时粒子逐渐消散
- 背景有绿色光晕随能量变化

## 乐曲库管理

### 添加歌曲
把音频文件放入 `D:\ai_workspace\hermes\music\`，重新打开播放器自动扫描。也可以点 `ALL` 然后切回 `↻` 刷新。

### 文件名规范
```
GAI周延 - 江湖流.ogg             → 艺人: GAI周延, 歌名: 江湖流
C-BLOCK _ GAI周延 - 江湖流.ogg    → 艺人: C-BLOCK / GAI周延, 歌名: 江湖流
```

QQ音乐下载的 `_H` `_M` `_L` 后缀自动去除。

### 自动切歌
当前曲目播完后自动跳到下一首，循环播放。

## 数据库

```
D:\ai_workspace\hermes\music\music-exe\music.db
```

与 exe 在同一目录，包含：
- 曲目信息（文件名、艺人、歌名）
- 播放历史（播放/喜欢/跳过记录）
- 每日推荐缓存

删除这个文件会清空所有历史和偏好，下次启动自动重建。

## 常见问题

**Q: 没有声音？**
A: 检查Windows音量、扬声器。确保音频文件格式受支持。WSL环境下测试不会有声音——播放器只在Windows本机有音频输出。

**Q: 双击exe没反应？**
A: exe 是后台GUI程序，会在任务栏出现窗口。如果闪退，在命令行运行 `Music-player.exe` 查看错误信息。

**Q: 启动后没有歌曲？**
A: 确保 `D:\ai_workspace\hermes\music\` 目录下有支持的音频文件。点 `ALL` 按钮查看扫描结果。

**Q: mflac 文件播不了？**
A: 播放器尝试用 QMCv2 算法解密。如果解密失败，文件可能是新版加密格式。可以先用 QQ音乐 转换成标准格式。

**Q: 如何开机自启？**
A: 右键 `Music-player.exe` → 创建快捷方式 → 放到 `shell:startup`（Win+R 输入 shell:startup）。

**Q: 之前 Hermes 版的记录还在吗？**
A: 在，两个播放器共用同一个 `music.db`，历史记录和偏好都通。

**Q: 点 ♥ 后闪红色但不恢复？**
A: v2.1.2 已修复——保存原始样式并在动画后恢复。

**Q: 同一首歌出现两次推荐？**
A: 曲库里同时有 `.ogg` 和 `.flac` 两个版本时会导致重复。v2.1.3 推荐引擎已按"艺人+歌名"去重，同首歌只出现一次。

**Q: 底部统计显示的是什么？**
A: 显示的是当前正在播放这首歌的播放次数/喜欢/跳过，不是全局总数。切歌自动更新。

**Q: 全局总数在哪里看？**
A: 顶栏右侧，时间旁边。格式：`▶ 5  ♥ 2  ⏭ 1`

## 技术栈

| 组件 | 技术 |
|---|---|
| UI | PyQt5 (Fusion 样式) |
| 音频引擎 | pygame.mixer |
| 数据库 | SQLite (自包含) |
| 动画 | QPainter + QTimer (30fps) |
| 加密解密 | QMCv2 XOR 算法 |
| 图标 | 均衡器立柱设计（Pillow 矢量生成 .ico） |
| 打包 | PyInstaller (单文件 exe，--icon 嵌入图标) |
