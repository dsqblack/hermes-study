---

## 一、现状分析

### 项目概况
Hermes Study 是 Spring Boot 3.2 + MyBatis 个人学习项目，含两个功能模块：
- **股市情报站**（/stock）：A股行情采集展示
- **Memo Board**（/memo）：Markdown 备忘录管理

### 现有设计优点
| 维度 | 现状 |
|------|------|
| 配色 | 使用 CSS Variables，体系完整 |
| 字体 | Inter/PingFang SC 混排，中英文均佳 |
| 布局 | Memo 用瀑布流，Stock 用表格，选型合理 |
| 响应式 | 有 media query 断点，移动可用 |
| 交互 | Hover 状态、Loading 反馈、Error 处理均有 |

### 现有设计痛点
| 维度 | 问题 |
|------|------|
| 视觉一致性 | 三个页面 radius 不统一（3px / 8px / 10px 混用） |
| 导航 | 各页面无返回首页入口，模块间切换靠浏览器后退 |
| 首页 | 仅导航卡片，无数据摘要/快捷入口 |
| Stock 页 | 必须手动点"开始收集"，无自动刷新 |
| Memo 页 | 缺少批量操作、拖拽排序；星级展示信息密度低 |
| 暗色模式 | 完全未支持 |
| 无障碍 | 无 ARIA role，颜色对比未验证 WCAG AA |

---

## 二、设计原则

```
极简 · 一致 · 数据优先
```

1. **极简**：去掉装饰性视觉噪音，让数据说话
2. **一致**：统一 Design Token，三页面用同一套变量
3. **数据优先**：首屏直接展示关键数据，减少操作步骤

#### 关键改进点

| 改进项 | 现状 | 方案 |
|--------|------|------|
| 自动刷新 | 无 | 页面加载自动执行，右上角手动刷新按钮 |
| 指数卡片 | 纯数字 | 增加迷你走势条（sparkline，纯 CSS 伪元素） |
| 双栏布局 | 单列 | 板块表 / 个股表并排（桌面端），移动端堆叠 |
| 涨跌颜色 | --green/--red 命名混乱 | 统一用 --rise / --fall 语义命名 |
| 错误状态 | 页面级展示 | 改为 Toast 通知，不遮挡已有数据 |
| 导航 | 无返回 | Header 左侧加 ← 回首页 |

## 五、组件规范（统一所有页面）

### 按钮

```
主操作按钮:   bg=--blue  color=white  h=36px  px=18px  radius=--radius-md
次操作按钮:   bg=transparent  border=--border  h=36px  px=14px
危险按钮:     bg=--danger  color=white
图标按钮:     36×36px  radius=--radius-md  border=--border
```

### 输入框

```
border: 1px solid --border
border-radius: --radius-md
h=36px  padding=0 12px
focus: border-color=--blue; box-shadow=0 0 0 3px --blue-light
```

### 数字/行情展示

```
涨：color=--rise  bg=--rise-bg
跌：color=--fall  bg=--fall-bg
数字字体：--font-mono（tabular-nums）
```

### Toast 通知

```
底部中央弹出，3s 自动消失
成功: bg=--success / 失败: bg=--danger
width=320px  radius=--radius-md  shadow=--shadow-modal
```

## 六、响应式策略

| 断点 | 宽度 | 变化 |
|------|------|------|
| desktop | ≥1024px | Memo 左侧 Sidebar + 3-4 列瀑布流 |
| tablet | 768-1023px | Sidebar 收起为顶部 tab，2 列瀑布流 |
| mobile | <768px | 顶部 tab，1 列瀑布流，底部 FAB |

---

## 七、无障碍（WCAG AA）

- 所有颜色对比比率 ≥ 4.5:1（正文）/ 3:1（大号文字）
- --rise（#e05c3a 对 white）= 3.7:1 → 大号文字合规
- --blue（#2563eb 对 white）= 5.4:1 ✓
- 所有交互元素有 `:focus-visible` 样式
- 图标按钮补充 `title` 和 `aria-label`
- Memo 卡片列表使用 `role="list"` + `role="listitem"`
